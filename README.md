# E-Commerce Sales Dashboard

Interactive Power BI dashboard for analyzing e-commerce sales performance, customers, and products.

## Tools Used
- Power BI
- Excel
- DAX
- Power Query

## Features
- Total Sales Analysis
- Profit & Revenue KPIs
- Top Selling Products
- Customer Insights
- Sales by Category
- Monthly Sales Trends
